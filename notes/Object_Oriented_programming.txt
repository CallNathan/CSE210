Object Oriented Programming
Difference between Objects and Classes
Objects are the variables created by Classes
Classes are the code

Variable: any noun/thing (name age number)
Function: verb (do something)

Object: also a noun/thing   
    -combine variables and functions in a datatype (class)
    -objects create a class

Class: defines a datatype
    -such as INT, String, BOOL
